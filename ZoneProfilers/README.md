# ZoneProfilers.jl (WIP)

A Julia interface for the Tracy instrumentation-based profiler, designed for multithreaded simulations.

ZoneProfilers.jl provides utilities to markup code for logging and timing.

ZoneProfilerTracy.jl provides a wrapper of the Tracy C library for profiling marked up code.

Documentation: https://github.com/nhz2/ZoneProfilers.jl
